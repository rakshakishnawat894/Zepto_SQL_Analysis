🛒 Zepto Product Data Analysis using SQL

This project focuses on analyzing and cleaning product data from a Zepto-like inventory database using SQL. The goal is to explore product details, identify data issues, and generate useful business insights such as stock availability, pricing patterns, and discount trends.

📌 Project Overview

The dataset contains product-level information including category, pricing, discount, stock availability, and quantity. SQL queries are used to:

Explore the dataset
Clean inconsistent or incorrect data
Perform analysis to extract meaningful insights
